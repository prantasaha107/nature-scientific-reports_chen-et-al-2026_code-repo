# Codes used in "A Regulatory Network Informed Investigation of macrophage cAMP dynamics during Mycobacterium tuberculosis infection" 

This repo contains code used to generate results for the paper "A Regulatory Network Informed Investigation of macrophage cAMP dynamics during Mycobacterium tuberculosis infection" by Chris Chen, Pranta Saha, Joyce Reimer, Shaun Wachter, Jeff Chen, Gordon Broderick, and Neeraj Dhar.

## Contents
* Codes pertaining to construction of the network itself can be found in `Network_Assembly/` (Sections 2.1-2.3 in paper)
* For identification of network parameters as well as the simulation and perturbation routines, see `Network_Logic_Identification_and_Simulation/` (Sections 2.4-2.5)
